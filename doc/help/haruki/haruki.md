# HarukiBot 使用指南 <Badge type="info" text="经典版" />

::: info ℹ️ 使用须知
HarukiBot 指令**不需要** `@mzkbot` 前缀，直接发送即可。
:::

## 📖 功能概览

| 分类 | 功能模块 | 说明 |
| :--- | :--- | :--- |
| 🎵 **歌曲查询** | pjskinfo、谱面预览 | 歌曲信息、BPM、谱面查看 |
| 👤 **玩家查询** | 绑定、逮捕、个人信息 | 账号绑定、排位查询 |

| 🃏 **卡牌查询** | 查卡、查活动 | 卡牌、活动信息 |
| 🌱 **养成系统** | MySekai分析、地图 | 数据分析、资源管理 |

## 🎵 PJSK 歌曲信息相关查询

### pjskinfo 系列

| 指令 | 说明 |
| :--- | :--- |
| `pjskinfo<曲名>` | 查看当前歌曲详细信息（支持别名：`song`、`musicinfo`）|
| `pjskbpm<曲名>` | 查看当前歌曲的 BPM |
| `查bpm<数字>` | 查询对应 BPM 的所有歌曲 |

### 谱面预览

```bash
谱面预览 <曲名> <难度>
```

::: info ℹ️ 难度参数
支持的输入：`easy`, `normal`, `hard`, `expert`, `master`, `append`

也可使用缩写：`ez`, `nm`, `hd`, `ex`, `ma`, `ap`, `apd`

> **注意**：如果查询 `master` 难度可省略难度参数
:::

### 昵称设置

::: details ⚙️ 昵称管理指令

| 指令 | 功能 |
| :--- | :--- |
| `musicset<昵称>to<歌名>` | 设置歌曲昵称 |
| `musicdel<昵称>` | 删除歌曲昵称 |
| `charalias<昵称>` | 查看特定角色所有昵称 |
| `charaset<昵称>to<角色名>` | 设置角色群通用昵称 |
| `charadel<昵称>` | 删除角色群通用昵称 |
| `grcharaset<新昵称>to<已有昵称>` | 设置仅当前群可用角色昵称 |
| `grcharadel<已有昵称>` | 删除仅当前群可用角色昵称 |

:::

::: warning ⚠️ 日志公示
所有歌曲/角色昵称设置的日志内容将会在实时日志页面按日公示。若违反相关条例将会视情况采取删除对应昵称至禁止使用 Bot 不等的措施。
:::

## 👤 查询玩家信息

### 多服支持 <Badge type="tip" text="重要" />

在命令前添加前缀可切换服务器：

| 前缀 | 服务器 | 示例 |
| :--- | :--- | :--- |
| (无) | 日服（默认） | `绑定` |
| `en` | 国际服 | `en绑定` |
| `tw` | 台服 | `tw绑定` |
| `kr` | 韩服 | `kr绑定` |
| `cn` | 国服 | `cn绑定` |

### 基础指令

```bash
绑定+id  # 通过游戏 UID 绑定你的游戏账号
```

## 🌱 MySekai 相关查询

::: info ℹ️ 数据上传方式
- **Android 用户**：建议使用 [Haruki 工具箱 - 上传 MySekai 数据](https://haruki.seiunx.com/upload_mysekai) 的 `引继信息上传模式`
- **iOS / iPadOS 用户**：建议使用代理工具 MitM 模块更新

**前置条件**：
- 拥有代理工具（如 Shadowrocket / Quantumult X / Surge / Loon / Stash）
- 有需要请加群 **1026390204** 咨询群友

:::

### 分析与地图

| 指令 | 别名 | 功能说明 |
| :--- | :--- | :--- |
| `mysekai_analyze` | `msa` | 根据上传的数据分析 MySekai 现存材料 |
| `mysekai_analyze2` | `msa2` | 新版 UI 设计（感谢 Foglio 的设计）|
| `mysekai_maps` | `msm` | 生成资源分布图 |

::: details 🗺️ 地图参数
`mysekai_maps` 指令后面添加数字可以查询单图地图：

- 数字 1: 草原
- 数字 2: 花田
- 数字 3: 沙滩
- 数字 4: 废墟

:::

### 照片下载

```bash
mysekai照片 + 序号  # 或 msp + 序号
```
根据用户上传到 Haruki 数据库的数据下载用户在 MySekai 里面拍摄的照片（按拍摄顺序）

## 🏆 活动排名相关查询

### 数据采集精度 <Badge type="info" text="技术细节" />

| 服务器 | T10 精度 | T11-T100 精度 |
| :--- | :--- | :--- |
| **日服、国服** | 10 秒 | 1 分钟 |
| **台服** | 30 秒 | 1 分钟 |
| **韩服、国际服** | - | 1 分钟（统一）|

### World Link 支持

::: info ℹ️ WL 单榜查询
以下指令添加 `wl` 前缀可以查询 World Link 活动单榜的排名和分数。

指令最后添加 `-c <角色别名>` 可以获取特定角色单榜的排名和分数，不添加则获取**目前正在进行的**单榜排名和分数。

**示例**：
- `wlsk 100` — 获取**目前**单榜的 100 名的排名和分数
- `wlsk 100 -c haruka` — 获取**桐谷遥**单榜的 100 名的排名和分数

:::

### 核心指令

| 指令 | 别名 | 功能 |
| :--- | :--- | :--- |
| `sk` | - | 如果你在前 100，可以用该命令查询排名和分数 |
| `sk <游戏数字id>` | - | 查询特定游戏玩家的排名和分数 |
| `sk <排名>` | - | 查询特定排名玩家的排名与分数（支持同时查询最多 7 个玩家）|
| `查房 <排名>` | `cf` | 查询特定排名最近 1 小时相关信息 |
| `sk线` | - | 查询榜线分数 |
| `时速` | - | 查看近一小时各榜线的 PT 增长速度 |
| `半日速` | - | 查看近半天各榜线的 PT 增长速度 |
| `日速` | - | 查看近一天各榜线的 PT 增长速度 |

::: details ⏱️ 时速指令技巧
`时速`、`半日速`、`日速`指令后面可以跟数字，将特定时间范围内的 PT 增长转换为对应速度

如 `时速10` = 10 分钟内 PT 增长量转换的时速

可输入数字单位为分钟，最大不超过 1440 分钟（即一天）

:::

### 追踪功能

| 指令 | 别名 | 功能 |
| :--- | :--- | :--- |
| `分数线 <排名>` | `rtr` | 查看本期活动中特定排名的分数趋势 |
| `追踪` | `ptr` | 追踪**自己**在本期活动中的活动 PT 趋势与排名趋势 |
| `追踪 <排名>` | `ptr <排名>` | 追踪**目前特定排名**在本期活动中的趋势 |

::: info ℹ️ 双排名查询
`追踪`指令支持双排名查询  
如 `追踪1 2` 会对比目前排名为 1 和 2 的两个玩家的活动 PT 趋势和活动排名趋势

:::

## 🃏 卡牌及活动信息相关查询

> 查卡面/查活动功能为 [Yozora](https://github.com/cYanosora) 所写，非常感谢 ❤️

### 卡牌查询

| 指令 | 功能 |
| :--- | :--- |
| `查卡/查卡面/查询卡面/findcard [角色昵称]` | 获取当前角色所有卡牌 |
| `查卡/查卡面/查询卡面/cardinfo [卡面id]` | 获取卡牌 id 详细卡面信息 |

### 活动查询

| 指令 | 功能 |
| :--- | :--- |
| `查活动/查询活动/event [活动id]` | 查看指定活动信息（可直接使用 `event` 查看当前活动信息）|
| `查活动/查询活动/findevent [关键字]` | 通过关键字筛选活动，返回活动概要图 |

::: details 🔍 关键字筛选示例

**单关键字方式**：
- `查活动 5v5`：返回活动类型为 5v5 的活动概要
- `查活动 紫月`：返回活动属性为紫月的活动概要
- `查活动 knd`：返回当期出卡（包括报酬）含 knd 的活动概要
- `查活动 miku`：返回当期出卡含任意组合的 miku 的活动概要

**多关键字方式**：
- `查活动 草 5v5`：返回活动类型为 5v5，活动属性为绿草的活动概要
- `查活动 knd 蓝`：返回活动属性为蓝星，出卡含 knd 的活动概要
- `查活动 普活 紫月 knd`：返回活动类型为马拉松，活动属性为紫月，出卡含 knd 的活动概要

**使用例**：
- `查活动 25h`：仅返回 25 箱活的活动概要
- `查活动 25h 25miku`：返回 25 箱活且 vs 出卡为白葱的活动概要
- `查活动 knd ick`：返回出卡同时包括 knd、ick 混活的活动概要

:::

### 活动总览

```bash
活动图鉴/活动列表/活动总览/findevent all
```
返回当前所有活动的概要（该功能由于图片过大无法在频道 bot 使用）

## 💡 快捷参考

### 最常用
1. `pjskinfo <歌名>` — 查歌曲信息
2. `sk` — 查排名
3. `逮捕 @某人` — 查打歌情况
4. `谱面预览 <歌名> <难度>` — 看谱面

### 数据分析
1. `mysekai_analyze` (`msa`) — 材料分析
2. `mysekai_maps` (`msm`) — 资源地图
3. `追踪` / `ptr` — 排名追踪
