# 📂 开源

本 Bot 的部分核心功能模块已基于 **GPLv3 协议** 开源。欢迎开发者查阅源码、二次开发或集成至自己的项目中。

> [!tip]⚠️注意
> 在使用或修改代码时，请务必保留原作者的版权信息及声明。

## 🧩 AstrBot 插件系列
- **[PJSK 猜卡面](https://github.com/yonglanws/astrbot_plugin_pjsk_guess_card)**：Project SEKAI 角色卡片识别与猜测游戏。
- **[PJSK 猜曲绘](https://github.com/yonglanws/astrbot_plugin_pjsk_guess_jacket)**：根据歌曲封面猜测曲目。
- **[PJSK 歌词猜曲](https://github.com/yonglanws/astrbot_plugin_pjsk_guess_lyrics)**：通过歌词片段互动猜歌。
- **[AI 智能问答](https://github.com/yonglanws/astrbot_plugin_askyou)**：基于模式匹配的AI问答插件
- **[每日签到](https://github.com/yonglanws/astrbot_plugin_signin)**：用户签到系统。

## 🌐 Web 前端
