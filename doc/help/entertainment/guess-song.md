# PJSK 猜歌 <Badge type="warning" text="娱乐" />

## 📖 功能简介

根据音频片段猜测   Project SEKAI   中的歌曲名称，考验你的音乐知识储备！

::: info 🎵 游戏规则

- 系统随机播放一段歌曲音频（约10秒）
- 玩家需要在   30秒内   发送编号回答
- 每个玩家有   2次   作答机会
- 答对可获得积分奖励
  :::

## 🎯 使用方法

### 基本指令

```bash
@mzkbot pjsk猜歌
```
