# PJSK 歌词猜曲

## 使用方法

### 开始歌词猜曲

```
@mzkbot 歌词猜曲
```

### 查看分数

```
@mzkbot 歌词猜曲分数
```

### 查看排行榜

```
@mzkbot 歌词猜曲排行榜
```

### 自定义排行榜名称

```
@mzkbot 歌词猜曲自定义名称 自定义名称
```

## 注意事项

> [!warning] 限制说明
> 由于部分群聊使用频率过大导致 bot 经常被风控，现已限制其他群聊的娱乐功能使用，请移步至 mzkbot 的娱乐功能专群 **1098198311** 游玩。
