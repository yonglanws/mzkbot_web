# 高阶数据分析

> [!warning] 数据要求
> 以下功能**必须上传游戏数据**才能使用。请先完成 [数据上传](./user-data#数据上传)。

## 排位与评分

### 查看排位信息

```
rk
```

获取玩家当前排位信息。

### B39 分析

```
b39
```

获取 Rating 最高的 39 首歌曲及总分（参考用）。

### B30 分析

```
pjsk b30
```

获取 Rating 最高的 30 首歌曲及平均分（算法不同）。

## 歌曲进度查询

### Expert 进度

```
pjsk进度ex
```

查询 **Expert** 难度歌曲完成情况（AP/FC/Clear）。

### Append 进度

```
pjsk进度apd
```

查询 **Append** 难度歌曲完成情况。

## 详细信息查询

### 个人详细信息

```
pjskdetail
```

获取详细信息，包括金币、水晶、家具、每日挑战等。

> 介意隐私者可私聊查询。

### 卡牌图鉴

```
pjskcard
```

获取个人卡牌图鉴。

```
pjskcard+角色名
```

查指定角色的卡牌（如：`pjskcard+奏`）。

### 活动历史

```
pjskevent
```

获取个人历史最高活动排名记录（近 10 次）。

## 多服支持

所有指令都支持多服前缀：

```
# 日服（默认）
rk
b39

# 台服
twrk
twb39

# 国服
cnrk
cnb39
```
