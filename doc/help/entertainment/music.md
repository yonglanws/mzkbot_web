# 网易云点歌

## 使用方法

### 标准点歌

```
@mzkbot 点歌 歌名
```

**示例：**

```
@mzkbot 点歌 千本樱
```

### 自然语言点歌

```
@mzkbot 直接描述你想听的歌
```

**示例：**

```
@mzkbot 我想听胃弱的...
@mzkbot r-906 的歌超好听！mzk 能给我推荐几首吗
```
