# PJSK 歌词猜曲 <Badge type="warning" text="娱乐" />

## 📖 功能简介

根据**歌词片段**猜测对应的 PJSK 歌曲，真正的歌词达人挑战！

::: info 🎤 游戏规则

- 系统展示一段歌词（中日双语）
- 提供 **10个选项**供选择
- 玩家需要输入对应**数字编号**作答
- 每位玩家最多可回答 **2次**，全局共 **6次** 机会
  :::

## 🎯 使用方法

### 基本指令

```bash
@mzkbot 歌词猜曲
```
