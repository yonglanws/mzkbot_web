# 每日签到 <Badge type="info" text="每日" />

## 📖 功能简介

每日签到系统是 mzkbot 的基础功能之一，通过每日签到可以抽取今日运势，增加互动性和趣味性。

## 🎯 使用方法

### 基本指令格式

```bash
# 方式一：@机器人
@mzkbot 签到

# 方式二：直接发送
签到
```
