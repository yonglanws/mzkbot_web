# PJSK 猜曲绘 <Badge type="warning" text="娱乐" />

## 📖 功能简介

根据歌曲封面（曲绘）的**局部截图**猜测对应的歌曲名称，测试你的 PJSK 曲绘识别能力！

::: info 🎨 游戏规则

- 系统展示曲绘的随机裁切部分
- 玩家需要在 **30秒内** 输入歌曲名称作答
- 支持中文名、原名、别名等多种回答方式
- 支持模糊匹配，降低答题难度
  :::

## 💡 答题技巧

::: tip 🎯 提示

- 观察曲绘的**配色方案**和**画风**
- 注意**背景元素**和**装饰图案**
- 记住热门歌曲的**标志性曲绘**
- 不确定时可以尝试**常用别名**
  :::
