# 基础对话 <Badge type="tip" text="AI" />

## 📖 功能简介

接入 **AI 模型**，为您提供真实的情绪价值与陪伴。mzkbot 能够理解上下文，进行自然流畅的对话交流。

::: info ✨ 核心特性
- 🤖 **智能对话** — 基于 AI 模型的自然语言处理
- 💬 **情感陪伴** — 提供情绪价值，支持闲聊和倾诉
- 📚 **知识问答** — 回答各类问题和提供信息
:::

## 🎯 使用方法

### 基本指令格式

```bash
@mzkbot + 您想说的内容
```

## ⚙️ 高级用法

| 场景 | 示例指令 | 说明 |
| :--- | :--- | :--- |
| 日常问候 | `@mzkbot 早安` | 获取温暖的问候语 |
| 情感倾诉 | `@mzkbot 今天心情不好...` | 获得安慰和建议 |
| 知识问答 | `@mzkbot 什么是PJSK` | 获取相关信息 |
| 闲聊互动 | `@mzkbot 讲个笑话` | 轻松娱乐 |

## ⚠️ 注意事项

::: warning ⏰ 响应时间
在使用高峰期，对话功能可能会出现较为严重的延迟，请耐心等待。
:::

::: tip 💡 使用建议
- 避免发送过长消息，可能导致响应缓慢
- 如遇到无响应，可稍后重试
- 尊重 AI 对话边界，避免敏感话题
:::

## ❓ 常见问题

::: details Q1：为什么有时候回复很慢？
高峰期用户较多，服务器负载较大，可能导致延迟增加。建议错峰使用或耐心等待。
:::
::: details Q2：可以连续对话吗？
支持上下文理解，可以在短时间内进行连续对话。
:::
