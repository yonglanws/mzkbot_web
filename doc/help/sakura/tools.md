# 查询与工具箱

> [!info] 使用说明
> 以下功能**无需上传数据**，绑定游戏 ID 后即可使用。

## 活动与排名

### 活动查询

```
sk
分数线
sk线
```

查询活动排名或指定档线分数。WL 活动支持加角色名查单榜。

### 档线时速

```
ss
```

计算当前档线时速（20 分钟速度 × 3）。

```
wlss
```

计算角色单榜档线时速（日服限定）。

### 查房

```
查房
```

查看本人或目标档线最近 1 小时活动情况。

### 活动信息

```
活动 [编号]
```

查看当前活动信息，加编号查历史（如：`活动 86`）。

```
findevent [参数]
```

高级活动查询，`findevent help` 查看用法。

### 5v5 活动

```
5v5 胜率
5v5 分数
```

查询 5v5 活动双方胜率及结算分数。

```
tf启动
```

5v5 活动中赠送无偿火（功能维护中）。

## 歌曲与谱面

### 歌曲信息

```
pinfo [歌曲名]
```

查询歌曲信息，支持花名。

**花名管理：**

```
pset 新昵称 to 旧昵称  # 设置花名
pdel 昵称              # 删除花名
```

### 难度排行

```
难度排行 [数量] [属性]
```

支持 `fc`/`ap` 前缀。

**示例：**

```
难度排行 10 紫
fc难度排行 10 紫
ap难度排行 10 紫
```

### 谱面预览

```
[歌曲名] 谱面预览
```

默认 Master，支持 `ex`/`apd` 后缀。

**示例：**

```
独りんぼエンヴィー 谱面预览
独りんぼエンヴィー 谱面预览ex
独りんぼエンヴィー 谱面预览apd
```

### 技能预览

```
[歌曲名] 技能预览
```

查看谱面技能分布，支持 `ex`/`apd` 后缀。

### 歌曲搜索

```
查bpm [最小值] [最大值]
查物量 [数量]
随个 [组合] [level]
葱什么
```

## 卡牌查询

### 卡牌搜索

```
findcard [参数]
查卡 [参数]
```

多条件筛选卡牌，最多 3 个参数（队名/角色/稀有度/属性）。

**示例：**

```
findcard ln 4星 粉
查卡 奏 4星
```

### 卡面查看

```
card [编号]
```

查看卡面图。

```
cardinfo [编号]
```

查看卡牌基础信息。

```
看 [角色名]
```

随机返回该角色卡图。

## 小游戏

### PJSK 听歌识曲

```
pjsk听歌识曲
```

随机播放片段，40 秒内回答。

```
结束猜曲
```

中断当前游戏。

### PJSK 猜卡面

```
pjsk猜卡面
```

随机截取卡面局部，40 秒内回答。

```
结束猜卡面
```

中断当前游戏。

### PJSK 抽卡

```
pjsk抽卡        # 默认 10 连图片
pjskXX连        # 文字结果（如：pjsk100 连）
pjsk反抽卡      # 反转概率，需指定卡池
```

### PJSK 表情制作

```
pjsk -h
```

查看详细说明。

## 其他工具

### 角色信息

```
charinfo
```

查询角色所有花名。

```
charset [昵称] [角色]
chardel [昵称]
```

修改角色花名。

### 自动推送

> 每日 **10:00** 自动检测更新并推送至订阅群，活动期间推送结活 Live 通知。
