# PJSK 猜卡面 <Badge type="warning" text="娱乐" />

## 📖 功能简介

根据角色卡面的的的局部图片的的猜测对应的角色名称，考验你对 PJSK 角色卡面的熟悉程度！

::: info 🎴 游戏规则

- 系统展示一张卡面的局部（分块打乱或裁切等）
- 玩家需要在   30秒内   发送角色名称缩写作答
- 支持多种难度模式
- 答对可获得积分奖励（根据难度不同）
  :::

## 🎯 使用方法
