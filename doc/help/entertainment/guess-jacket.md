# PJSK 猜曲绘

## 使用方法

### 开始猜曲绘

```
@mzkbot 猜曲绘
```

### 查看分数

```
@mzkbot 猜曲绘分数
```

### 查看排行榜

```
@mzkbot 猜曲绘排行榜
```

### 自定义排行榜名称

```
@mzkbot 猜曲绘自定义名称 自定义名称
```
