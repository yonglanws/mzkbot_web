# 使用 BOT 的规范 <Badge type="danger" text="必读" />

::: tip ⚠️ 重要提醒
请仔细阅读并遵守以下规则，共同维护良好的 BOT 使用环境。违反规则可能导致功能限制或封禁。
:::

## 🚫 红线规则（禁止行为）

::: danger 严重违规
以下行为将直接导致 **永久封禁**，恕不另行通知。
:::

| 序号 | 行为 | 说明 |
| :--- | :--- | :--- |
| 1 | 🚫 **不准辱骂 BOT** | 对 BOT 使用不尊重的言语，包括但不限于辱骂、人身攻击等行为 |
| 2 | 🚫 **不准刷屏** | 在群聊中频繁调用 BOT 发送大量消息，造成群聊混乱或干扰他人 |
| 3 | 🚫 **不准间接表达不适言论** | 通过 BOT 间接发布包含敏感、歧视、引战或令人不适的内容 |
| 4 | 🚫 **不准拉入无关群聊** | 将 BOT 添加至与 PJSK 无关的群聊（例如无关社交群等） |

## ⚠️ 黄线规则（需注意行为）

::: warning 注意事项
以下行为可能导致功能限制或风控，请务必注意。
:::

| 序号 | 行为 | 风险说明 |
| :--- | :--- | :--- |
| 1 | ⚠️ **避免高频操作** | 短时间内发送大量请求可能导致 BOT 触发风控，影响正常使用体验 |
| 2 | ⚠️ **不要禁言 BOT** | 禁言 BOT 可能导致腾讯风控系统判定异常，影响服务稳定性 |
| 3 | ⚠️ **不要主动踢出 BOT** | 如需退群，请联系 BOT 管理员手动处理。直接踢出可能触发风控，影响后续使用 |

***

## 📋 违规处理说明

::: details 处理标准
根据违规情节轻重，将采取以下处理措施：

| 违规程度 | 处理方式 |
| :--- | :--- |
| 轻度 | 提醒警告 |
| 中度 | 临时封禁（1-7天） |
| 重度 | 永久拉黑 |

:::
