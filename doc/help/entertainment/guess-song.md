# PJSK 猜歌

## 使用方法

### 普通猜歌

```
@mzkbot pjsk猜歌
```

### 倍速猜歌

```
@mzkbot pjsk猜歌1    # 1.5 倍速
@mzkbot pjsk猜歌2    # 2 倍速
@mzkbot pjsk猜歌3    # 3 倍速
```

### 查看排行榜

```
@mzkbot 本地猜歌排行榜
```

## 注意事项

> [!warning] 限制说明
> 由于部分群聊使用频率过大导致 bot 经常被风控，现已限制其他群聊的娱乐功能使用，请移步至 mzkbot 的娱乐功能专群 **1098198311** 游玩。
