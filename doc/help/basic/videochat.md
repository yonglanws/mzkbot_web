# 视频对话 & 剧本生成 <Badge type="tip" text="AI" /> <Badge type="danger" text="新" />

## 🎬 视频对话

### 使用方法

```bash
@mzkbot 视频对话 你想要对他说的内容
```

### 使用示例

```bash
@mzkbot 视频对话 你好
```

::: tip 💡 说明
和文字对话类似，bot 会根据你的输入内容生成一段视频回复。使用方式与文字对话相似，只不过是以**视频的形式**回复。
:::

## 📝 剧本生成

### 使用方法

```bash
@mzkbot 剧本生成 剧本的主要内容
```

### 使用示例

```bash
@mzkbot 剧本生成 晓山瑞希吃薯条
```

::: tip 💡 说明
Bot 会根据你提供的主要内容，生成一段 **2-3 分钟** 以**晓山瑞希**为视角的视频。
:::

## ⚠️ 重要提示

::: danger 🚫 注意
- 已接入**违禁词检测系统**，请不要使用此功能进行违规操作。如发现滥用行为，将有权封禁相关权限。
- 请勿利用此功能进行盈利制作
:::

::: warning ⏰ 生成速度说明
- 视频生成所需时间较长，已尽力优化
- 已专门制作**排队系统**，请耐心等待
- 两个功能目前还不太稳定，如有问题请联系开发者
:::

::: tip 💡 使用建议
- 高峰期可能需要排队等待
- 请避免发送过于复杂或敏感的内容
- 尊重 AI 生成边界，避免违规话题
:::
