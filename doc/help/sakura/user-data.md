# 用户数据管理

> [!important] 使用须知
> SakuraBOT 指令**不需要** `@mzkbot` 前缀，直接发送即可。

## 基础指令

### 绑定游戏账号

```
绑定 ID
```

绑定你的 PJSK 游戏账号 ID。ID 可在游戏个人信息页查看。

### 设置查询权限

```
给看
```

开放他人 `@` 查询你的权限。

```
不给看
```

关闭他人 `@` 查询你的权限（默认）。

### 查看个人资料

```
pjskprofile
```

获取个人基本信息卡片。

### 查看当前绑定

```
id
```

查看当前 QQ 绑定的游戏 ID。

## 社交查询

### 查看他人队伍

```
视奸 @用户
```

查看指定用户的当前队伍综合力及构成。需要对方开启「给看」权限。

### 查询歌曲进度

```
逮捕
```

查询 Expert/Master FC/AP 进度。**需先上传数据**。

## 数据上传

以下高阶功能（如 `rk`, `b39`, `进度` 等）**必须上传游戏数据**才能使用。

1. 访问数据上传页面：[点我进入](http://go.mikuware.top/)
2. **日服玩家特权**：可直接复制游戏内的「引继信息」发送给 Bot 自动完成上传

> 需注意引继码一次性失效，请谨慎操作或重新引继。
